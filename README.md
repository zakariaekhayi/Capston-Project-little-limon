About the Project
This project was developed as part of the Meta Coursera Front-End Development Course, marking the culmination of the Front-End Development track.

The website demonstrates a table reservation system for Little Lemon restaurant, implemented using React components to highlight understanding of component-based design and state management. The project also incorporates API requests to handle dynamic data.

Important Note: At this stage, the main interactive functionality available is the Reserve a Table feature; other functionalities are currently static.
